Time 2 Track Animated Logos

Time 2 Track Animated Logos

Using the same logo presentation construct that I created for My Internship Journal I decided to try my hand at animating some logos. When you arrive at the page the first logo animates onto the screen. Upon scrolling or clicking to the next option the original logo resets itself and the next option’s animations are triggered. I use HTML Canvas, SVG path offsets, CSS animations and keyframes and some good old fashioned JavaScript to make the magic happen. I’m pretty happy with how this turned out and like to show it off as an example of web animation. I learned a lot in the process of making this.
