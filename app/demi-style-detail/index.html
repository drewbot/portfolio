Demi Interactive Style Guide

The Iron Yard hired me on contract to help out with an internal tool theyâ€™re working on. The Application will give teachers and administrators a place to review teacher contribution efforts and student evaluation in order to test effectiveness of curriculum and teaching methods. It will also serve as a hub for students to submit their work and review evaluations. This interactive style guide takes a stab at a dashboard of sorts as well as showing a few different layout implementations. I built this using Ember CLI, Bootstrap and Sass at the request of The Iron Yard and did so in a modular way so that the team could cherry-pick modules and components to fit their needs as development evolves. I used handlebars templates to isolate mark-up and incorporated as many Sass variables and mixins as I could, in order to make the style guide a useful tool for the next developer to continue the project. There are several other components to this project that I worked on including a data collector for administrators to upload goals, resources and assignments which you can see here, a command line interface and a curriculum planner which you can see a sneak peek of here.
