Demi Data Collector

An independent component of the curriculum management app shown in style guide format here, The data collector is an entry point for goals, resources and assignments to be uploaded for later use in The Iron Yardâ€™s overall curriculum management system. This app is built in Backbone.js using Bootstrap and Sass and is connected to a Rails API. Like the style guide, the collector is responsive for accessibility by administrators on the go. I incorporated components and mixins from the style guide and used colors from the overall palette in a way that makes sense for the context, establishing the beginnings of a visual language. This app is in beta right now and only a few of the routes are firing so please be patient when loading and clicking around.
